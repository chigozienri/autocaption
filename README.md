# Cog ⚙️ Autocaption
This is a [https://cog.run](Cog) implementation of [https://github.com/fictions-ai/autocaption](autocaption).


##  Installation

```bash
    git clone https://github.com/chigozienri/autocaption
    cd autocaption
```
##  Usage

```bash
    cog predict -i video_file_input=@path/to/video.mp4
```
